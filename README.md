## Run Locally

Clone the project

```bash
  git clone https://github.com/orlocdiaz/holocruxe_team.git
```

Go to the project directory

```bash
  cd holocruxe_team
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
